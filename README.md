# RagNext Game Designer
> **PREMIUM STORY ENGINE & RPG PLAYER**

Welcome to the official public repository for the **RagNext Game Designer** documentation and release distribution. 

RagNext Game Designer is the official cross-platform visual scripting and world editor for the RagNext Game Engine. It features intuitive node-graph compilation, dialogue trees, and local LLM/AI workflow integration.

---

* **Interactive Comments**: Share feedback, ask questions, or report issues directly on each page.

## 🚀 Key Features

* **Visual Node-Graph Editor**: Design complex game logic and branching stories without writing code.
* **Dialogue Trees**: Write interactive conversations with dynamic condition checks.
* **World Editor**: Manage rooms, items, characters, and global state variables.
* **AI Workflow Integration**: Native support for local LLM models to assist in narrative generation.
