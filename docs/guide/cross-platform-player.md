# Cross-Platform Runtime Player

The **RagNext Player** is the companion runtime engine designed to load and execute story packages created inside the Game Designer. 

To ensure maximum accessibility and compatibility, the player is engineered from the ground up to be completely cross-platform.

---

## 🎮 Powered by Unity

By leveraging the power of **Unity**, the RagNext Player compiles your adventure games to run seamlessly across all major platforms:

* **Desktop**: Full native support for Windows, macOS, and Linux.
* **Mobile**: Optimized layouts and touch controls for iOS and Android.
* **Web**: Lightweight WebGL builds that can be played directly inside any modern web browser.

## 🖥️ Player Interface

The player features a clean, responsive, and distraction-free user interface designed to keep players focused on the story and gameplay:

* **Rich Multimedia Canvas**: Renders high-quality room artwork, layered composite images, and immersive background audio or sound effects.
* **Dialogue & Narratives**: A scrolling story feed displaying narrative text, descriptions, and dynamic variable values.
* **Interactive Choices**: Clean popup dialogs and choices that branch the script logic in real-time.
* **Status Bar & Side Panels**: Visible navigation directions, lists of loose objects/items in the room, and active character details.
